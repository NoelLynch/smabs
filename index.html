smabs

SMall Ass Build System

Simple build system that compiles, uglifies, minifies and assembles.

Why not just use gulp, grunt, cake etc...yeah I ask myself that a lot :-) I wanted something that felt more anty or maveny. So why not, it only took a few hours to write so feck it.

install

sudo npm install -g

usage

In any directory with a build.json file just run smabs, done.

build.json

Defines execution tasks and parameters.
